# Clavier macos

**Note : touche `option` == touche `alt`**

- `\` => `alt shift /` ou `alt option /`
- `~` => `alt n`
- `|` => `alt shift L` ou `alt option L`
- `[` => `alt shift (` ou `alt option (`
- `{` => `alt (`
- pour taper une backtick, il y a une touche dédiée
