# Var dumper

## Installation

    composer require symfony/var-dumper

## Utilisation

`hello-var-dumper.php` :

    <?php

    require __DIR__.'/vendor/autoload.php';

    $myVar = 'Hello Var Dumper';

    dump($myVar);
